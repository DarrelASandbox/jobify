## Installation

1. Install NPM packages.

```sh
npm install
```

2. Create .env file in Jobify folder.

```sh
touch .env
```

3.Enter the information in .env file

```env
PORT=4000
MONGO_URL= connection string into your application code from https://www.mongodb.com/atlas/database
```
